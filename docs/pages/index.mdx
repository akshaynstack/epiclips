# Epiclips

Welcome to **Epiclips** - an open-source AI-powered video clipping tool that transforms long-form videos into viral short-form clips.

## What is Epiclips?

Epiclips automates the process of creating short-form content from long videos like podcasts, tutorials, and streams. It uses:

## Features Overview

| Feature | Description |
|---------|-------------|
| AI Clipping | Automatically identifies viral moments |
| Face Tracking | Smooth, dynamic face following |
| Split-Screen | Screen + webcam layout for tutorials |
| Podcast Mode | 2-person vertical stacking |
| Caption Styles | 5+ presets including viral gold, neon pop |
